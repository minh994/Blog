Cài đặt Flutter trên Windows

Minh Le |   01 January 1010

Flutter là một framework phát triển ứng dụng di động mạnh mẽ, hỗ trợ xây dựng ứng dụng đa nền tảng từ một mã nguồn duy nhất. Trong bài viết này, chúng ta sẽ cài đặt Flutter trên Windows một cách chi tiết.

1. Tải xuống Flutter SDK

  1. Truy cập Flutter releases.
  2. Tải xuống tệp ZIP của Flutter SDK mới nhất.
  3. Giải nén tệp ZIP vào thư mục bạn chọn, ví dụ: 
    C:\flutter

2. Cấu hình PATH cho Flutter

Thêm Flutter vào biến môi trường Path:

  • Mở Control Panel > System > Advanced system settings > Environment Variables.
  • Tìm và chỉnh sửa biến Path.
  • Thêm đường dẫn sau vào danh sách: 
    C:\flutter\bin

Kiểm tra cấu hình bằng cách mở Command Prompt và gõ:

flutter --version

3. Kiểm tra cấu hình Flutter

Chạy lệnh sau để kiểm tra trạng thái cài đặt Flutter:

flutter doctor

Kết quả mẫu:

Doctor summary (to see all details, run flutter doctor -v):
[✓] Flutter (Channel stable, 3.7.0, on Microsoft Windows [Version 10.0.19045], locale en-US)
[✓] Android toolchain - develop for Android devices (Android SDK version 31.0.0)
[!] Chrome - develop for the web (Cannot find Chrome executable at .)
[!] Visual Studio - develop for Windows (Visual Studio not installed)
[!] Android Studio (not installed)
[✓] Connected device (1 available)

Cài đặt các công cụ còn thiếu theo hướng dẫn của flutter doctor.

4. Cài đặt Android Studio

Tải xuống và cài đặt Android Studio.

Trong Android Studio:

  • Cài đặt Android SDK thông qua SDK Manager.
  • Kích hoạt Android Virtual Device (AVD) để tạo trình giả lập.
  • Kiểm tra Android SDK:
flutter doctor --android-licenses

Chạy lệnh flutter doctor để kiểm tra lại cấu hình.

5. Chạy ứng dụng Flutter đầu tiên

Tạo một dự án Flutter mới:

flutter create my_first_app
cd my_first_app

Chạy ứng dụng trên trình giả lập hoặc thiết bị thực:

flutter run

Nếu cần cài đặt thêm trình giả lập:

flutter emulators --create
flutter emulators --launch <emulator_id>

Lưu ý

Kết nối thiết bị Android thực:

  • Bật Developer Mode trên thiết bị Android.
  • Kết nối qua cáp USB và bật USB Debugging.
  • Kiểm tra thiết bị kết nối:
flutter devices

Bạn có thể xem thêm chi tiết tại Tài liệu chính thức của Flutter.

flutter development blog
